# Project Chat

Project Chats provide a dedicated real-time space to brainstorm ideas, plan features, or debug issues with an AI agent specifically tailored to your project. 

## Real-Time Collaboration

Each project has a dedicated chat interface that uses isolated containers for live, multi-turn interaction. This means the AI maintains context of your conversation, allowing for deep dives into complex problems.

### Key Features

*   **Context Preservation**: The AI remembers previous messages in the conversation, making it easy to iterate on ideas.
*   **Image Support**: You can upload screenshots or diagrams (via the upload button) to provide visual context for your requests.
*   **Live Interaction**: Experience fast, responsive messaging powered by HTMX and ephemeral containers.
*   **Persistent History**: All conversations are automatically saved and synced, so you can pick up right where you left off.

## Moving from Chat to Build

Once you've refined an idea or a plan in the chat, you can easily turn it into an engineering task:

1.  **Refine**: Discuss the feature or fix until you're satisfied with the approach.
2.  **Build**: Use the **Build** button (or similar action) within the chat interface to submit your final prompt as a background Job.
3.  **Monitor**: Switch to the **Jobs** tab to watch the AI worker implement the changes discussed in the chat.

### Managing Chats
You can create new chats, delete old ones, or restore them from the trash using the chat sidebar.
