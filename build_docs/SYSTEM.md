# System Settings

The System Settings screen provides information about the overall health of the overdrive system and administrative actions.

## System Health

You can find information about the health of the overdrive system:

*   **API Uptime**: How long the overdrive API has been running.
*   **Scheduler Uptime**: How long the job scheduler has been running.
*   **Active Containers**: The number of currently running job or chat containers.

## Appearance

overdrive supports a variety of color schemes to suit your preference. You can change the theme using the **Theme Selector** in the sidebar. We offer 11 built-in schemes, including:

*   **Dark Themes**: Dracula, Monokai, Nord, Synthwave '84, and more.
*   **Light Themes**: Solarized Light and others.

You can also create your own custom theme to perfectly match your environment.

## Danger Zone

The **Danger Zone** contains actions that can affect the availability of the overdrive service:

*   **Rebuild and Restart API**: Recompiles the API server and restarts it.
*   **Rebuild and Restart Scheduler**: Recompiles the job scheduler and restarts it.

Use these actions only if you encounter persistent issues with the service.
