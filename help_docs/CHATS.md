# Project Chat

Project Chats provide a dedicated real-time space to brainstorm ideas, plan features, or debug issues with an AI agent specifically tailored to your project.

## Conversation and Context

Each project has a dedicated chat interface that leverages isolated containers for live, multi-turn interaction. This environment ensures the AI maintains a deep understanding of your conversation as it progresses.

### Key Features

*   **Context Preservation**: The AI remembers previous messages, allowing you to iterate on complex ideas through natural conversation.
*   **Image Support**: Upload screenshots or diagrams to provide visual context for your requests.
*   **Persistent History**: All conversations are automatically saved and synced, making it easy to resume work across different sessions.
*   **Real-Time Responsiveness**: Powered by HTMX and ephemeral containers for a fast and interactive messaging experience.

## Workflow Integration

Once you have refined a plan in the chat, you can transition it directly into an engineering task:

1.  **Refine**: Discuss the feature or bug fix until you're satisfied with the proposed approach.
2.  **Build**: Use the **Build** button within the chat interface to submit your final prompt as a background Job.
3.  **Monitor**: Switch to the **Jobs** tab to watch the AI worker implement the changes discussed.

## Managing Chats

The chat sidebar allows you to organize your conversations:

*   **Create**: Start new chat sessions for different topics.
*   **Rename**: Label your chats to easily find them later.
*   **Delete and Restore**: Move old conversations to the trash or restore them if needed.
